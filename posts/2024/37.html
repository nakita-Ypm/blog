React ​

  • Build a React App
sh
deno run -A npm:create-vite@latest

Directory Structure ​

.
└── react
    ├── deno.lock
    ├── index.html
    ├── package.json
    ├── public
    │   └── vite.svg
    ├── src
    │   ├── App.css
    │   ├── App.tsx
    │   ├── assets
    │   │   └── react.svg
    │   ├── index.css
    │   ├── main.tsx
    │   └── vite-env.d.ts
    └── vite.config.ts

Start ​

sh
cd react
sh
deno task dev

Accesse Browser ​

http://localhost:5173/
