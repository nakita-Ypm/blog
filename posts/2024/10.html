Bun Workspaces ​

 Bunのworkspacesを試す。

  • Configuring a monorepo using workspaces with Bun

  • sample project

ディレクトリ構成 ​

.
├── package.json
├── packages
│   ├── a
│   │   ├── index.ts
│   │   └── package.json
│   └── b
│       ├── index.ts
│       └── package.json
└── src
    └── index.ts

packages ​

package.json ​

json
{
  "name": "@pkg/a",
}

index.ts ​

ts
export const pkga = (): string => {
  return 'パッケージAです'
}

src/index.tsから利用 ​

package.json ​

  • package.json private

 ルートパッケージを誤って後悔しないように、"private": trueをつけるのが慣例。

json
{
  "private": true,
  "workspaces": [
    "packages/*"
  ]
}
ts
import { pkga } from '@pkg/a'
import { pkgb } from '@pkg/b'

const resa: string = pkga()
const resb: string = pkgb()

console.log(resa)
console.log(resb)

実行結果 ​

パッケージAです
パッケージBです

注意点 ​

 もし、ディレクトリ変更した場合

├── package.json
├── packages
│   ├── a
│   │   ├── package.json
│   │   └── src
│   │       └── index.ts
│   └── b
│       ├── package.json
│       └── src
│           └── index.ts
├── README.md
└── src
    └── index.ts

 "main"のパスを合わせる。

package.json ​

json
{
  "name": "@pkg/a",
  "main": "src/index.ts"
}
